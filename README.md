# 夜莺社区

夜莺社区是一个面向 Web3 + AI 的开源社区，目标是让用户真正拥有数据主权，并以用户为中心完成数据治理与应用创新。我们致力于构建用户可控的超级 APP / 数字人生态。

当前愿景（可迭代）：让价值回到用户，用可控数据组装智能助手，沉淀可复用的个人知识与能力体系。
