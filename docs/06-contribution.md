# 参与方式与贡献路径

我们欢迎开发者、研究者、创作者与投资方参与共建。以下是可持续参与的入口与方式。

## 贡献方向
- **开发者**：协议、工具链、插件、应用集成
- **研究者**：方法论、评估指标、路线验证
- **创作者**：内容、案例、产品叙事与社区传播
- **投资方/伙伴**：场景共建、资源支持、生态协同

## 参与路径（建议流程）
1. 阅读 [`docs/00-overview.md`](00-overview.md) 了解整体方向
2. 选择兴趣方向（数据层/协议层/AI层/应用层）
3. 进入对应工作组或提出提案
4. 以任务形式交付成果并获得反馈

## 贡献流程（待补充）
贡献流程与评审标准将在后续单独发布。若需临时参与，可先在仓库 Issue 中说明目标与范围。

## 贡献原则
- 尊重用户主权与隐私优先
- 产出可验证、可复用的成果
- 文档与代码同步，保证可持续协作

## 新手入口
- 文档改进与术语校对
- 小型工具或脚手架
- 示例应用与教程
