# 文档导航

本目录用于清晰说明夜莺社区“是什么、在做什么、怎么做”。建议按不同角色选择阅读路径。

## 阅读路径

### 非社区成员（想快速了解）
- `docs/00-overview.md`：社区总览
- `docs/01-why-now.md`：为什么是现在
- `docs/04-products.md`：已落地产品矩阵
- `docs/08-faq.md`：常见问题
- `docs/10-origin.md`：社区起源与命名
- `docs/13-public-intro.md`：对外介绍（公开版）

### 社区成员（参与与共建）
- `docs/06-contribution.md`：参与方式与贡献路径
- `docs/05-governance.md`：协作与治理机制
- `docs/07-roadmap.md`：路线图
- `docs/11-economics.md`：经济模型（权益体系）
- `docs/14-member-handbook.md`：成员手册（协作版）

### 开发者/项目方（接入与落地）
- `docs/03-architecture.md`：技术路线与分层架构
- `docs/04-products.md`：产品矩阵与仓库
- `docs/12-appstore-entitlements.md`：应用商店权益与计费规范
- `docs/15-entitlements-openapi.yaml`：权益接口 OpenAPI 草案

## 文档结构
