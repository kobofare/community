# DIY 超级 APP（方向性案例）

> 这是一个基于夜莺社区能力的“方向性案例”，用于对外说明“我们在做什么、怎么做”。

## 目标
让用户可以自由组合 AI Agent、Bot 与工具，搭建自己的超级 APP，同时保留数据主权与可撤销授权。

## 角色与用户
- 个人用户：希望有一个可 DIY 的个人助手
- 创作者/开发者：希望快速组合工具并发布
- 小团队：希望搭建内部知识与协作工具

## 关键能力模块
- **身份与授权**：钱包插件 + UCAN 作为授权入口
- **数据资产**：数字仓库（WebDAV）存储与迁移
- **知识与记忆**：知识库（RAG）提供检索、记忆与工作流
- **智能执行**：数字人/Agent 执行任务
- **分发与生态**：应用商店上架与共享

## 使用路径（示例）
1) 用户安装钱包插件并创建身份
2) 授权个人数据到数字仓库
3) 接入知识库并建立个人知识体系
4) 组合数字人/Agent 与 Bot 工具
5) 在应用商店发布与复用

## 价值与差异化
- 数据不被平台锁定
- 授权可撤销、可迁移
- 能力可组合、可复用

## 下一步落地（建议）
- 做出一个“单人可用”的最小工作台
- 完成 1 个 Agent + 1 个 Bot + 1 个工具的组合示范
- 形成对外演示与可复用模板
