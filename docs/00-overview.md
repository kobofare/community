# 社区总览

夜莺社区是一个面向 Web3 + AI 的开源社区，目标是让用户真正拥有数据主权，并以用户为中心完成数据治理与应用创新。我们相信：只有当数据回到用户手中，AI 才能成为真正的“个人智能”，并持续服务于用户的长期利益。

## 我们是谁
- 互联网精神的实践者：自由、开放、合作、共享
- 去中心化自治组织方向：透明治理与贡献驱动
- Web3 + AI 的实践社区：面向数据主权与个人智能体
- 可编程社会的推动者：让任何人能组合工具与内容

## 我们解决的问题
- 用户数据被平台锁定，难以迁移、授权与复用
- AI 需要可信、可授权的数据来形成“对个人的理解”
- 数据孤岛导致协作、创新与收益难以沉淀

## 我们的愿景与理念
当前愿景（可迭代）：让价值回到用户，用可控数据组装智能助手，沉淀可复用的个人知识与能力体系。

- 用户数据主权：数据由用户掌控、授权、收益共享
- 透明治理：规则可验证、流程可追溯、结果可复用
- 开放协作：以开源为底座，鼓励共建与快速迭代
- 以用户为中心：AI 的目标是最大化用户利益，而非平台利益

## 技术路线（四层结构）
1. 数据层：以钱包插件作为入口，建立个人数据资产目录
2. 协议层：授权、溯源、收益分配与治理规则
3. AI 层：个体认知模型与可协作数字人
4. 应用层：超级 APP / 数字人应用生态

## 已落地的产品矩阵
- 钱包插件：个人身份与数据入口
- DApp 应用商店：开放生态的承载与增长
- 数字仓库（WebDAV）：数据同步与资产管理能力
- 知识库：沉淀共识与工程实践
- 数字人应用：面试官数字人、督学老师数字人等场景化落地

## 我们如何协作
我们坚持开源共建，以透明治理为原则，采用提案与任务驱动的协作机制，鼓励以成果与贡献为核心的声誉体系，保证方向与建设可追踪、可复用、可迭代。

## 如何参与
面向开发者、研究者、创作者与投资方，我们提供项目协作、研究验证、产品孵化与生态共建的入口。参与方式与贡献路径见 `docs/06-contribution.md`。
