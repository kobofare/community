# 术语表

- **数据主权**：用户对自身数据拥有控制、授权与收益分配的权利
- **DAO（去中心化自治组织）**：以公开规则和社区共识驱动的自治协作组织形态
- **数据钱包**：承载个人身份与数据资产管理的入口工具
- **KY（Know Yourself）**：强调个体身份与自我认知的数字基础能力
- **数据资产目录**：对个人数据的结构化描述与索引
- **数据治理**：围绕数据授权、使用、溯源与收益分配的规则体系
- **授权**：用户对数据使用范围、期限与对象的明确许可
- **溯源**：数据与行为可追踪、可审计的能力
- **数字仓库**：面向个人数据的同步与资产管理能力
- **CP（贡献权益）**：贡献计分权益，用于治理权重与资源倾斜，不可转让
- **UC（使用权益）**：应用商店内可消耗的使用权益，不可转让/提现
- **权益包**：面向新用户/活动/邀请等场景的权益组合
- **试用权益**：限定额度与有效期的使用权益
- **可组合**：协议、工具与应用可重组、可迁移、可复用
- **可编程社会**：任何人可为自己与他人编程的数字-实体共生社会形态
- **数字人**：基于个人数据与模型构建的可协作 AI 角色
- **超级 APP**：聚合多场景能力、可组合扩展的用户应用
- **DApp 应用商店**：开放生态的应用分发与集成入口
- **个体认知模型**：描述用户偏好、目标与行为的可迭代模型
- **LOTC（链上生活）**：以链上合约驱动、围绕价值流转的生活场景形态
